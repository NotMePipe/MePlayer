A lightweight music player

  - Version 1
    - SDL window
    - play single song
  - Version 2
    - FFmpeg decoding pipeline
    - proper audio buffering
  - Version 3
    - metadata extraction
  - Version 4
    - library scanning
  - Version 5
    - album grouping
  - Version 6
    - UI polish
  - Version 7
    - playlists
  - Version 8
    - Discord RPC
    - Last.FM
